# Security Policy

## Supported Versions

Only the latest release of An Otter Wiki is supported. Due to resource constraints, we do not maintain support for older versions.

If you are using an older version, upgrading to the latest minor release is straightforward. See our upgrade guide: [https://otterwiki.com/Upgrade](https://otterwiki.com/Upgrade).

## Reporting a Vulnerability

> [!CAUTION]
> To report a potential security vulnerability or problem with An Otter Wiki, please **do not** open a public issue on GitHub.
> Instead, [submit a vulnerability report](https://github.com/redimp/otterwiki/security/advisories/new) or send an email to [security@otterwiki.com](mailto:security@otterwiki.com).

Please include the following information in your report:

- The version or image tag of An Otter Wiki release that is impacted
- How to reproduce the vulnerability (step-by-step)
- The platform / environment it occurs on
- Any potential fixes or reference code you think might be helpful in resolving the issue
- Your GitHub username if you'd like to be acknowledged as a contributor


There's no reward for submitting a report, but we will acknowledge bug reporters in release notes.
We appreciate your contribution to making An Otter Wiki more secure.
